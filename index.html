Receita de Pudim

X X X X X X X X X X X X X X X

Ingredientes

  • 1 Lata de leite condensado
  • 3 ovos
  • 1 litro de leite
  • 1 xícara de açúcar

Modo de preparo

  • Tranforme o açúcar em calda
  • Forre uma tigela com o caldo
  • Bata os ingredientes no liquidificador
  • Asse em banho maria por 90 minutos
